# Echoes of the Hollow Star

A local-only retro dungeon crawler proof of concept built with React, Vite, three.js, Fastify, and SQLite.

## Scripts

- `npm install`
- `npm run db:seed`
- `npm run dev`
- `npm run test`
- `npm run build`

## Notes

- The app uses a small authored dungeon and SQLite-backed save slots.
- Visual asset paths are wired for raster art under `client/public/assets`.
- If higher-fidelity art is generated later, keep the same filenames to swap it in without code changes.
